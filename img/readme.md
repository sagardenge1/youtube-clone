# Youtube-Clone #
## Description ##
This is a clone of the popular video-sharing platform YouTube, built using HTML, CSS and Bootstrap.

# Technologies Used #
1 HTML5  - used for structuring the web pages.
2 CSS3 
3 Bootstrap
